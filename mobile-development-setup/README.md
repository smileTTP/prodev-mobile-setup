# prodev-mobile-setup

The objective of this project is to explore mobile development with React Native, TypeScript and NativeWindCSS (using Expo Framework)

## Tools 

- OS (Windows)
- VS Code
- Node.js (v24.11.0)
- Expo CLI (npm install -g expo-cli)
- Expo Go [installed on Android](https://play.google.com/store/apps/details?id=host.exp.exponent)

## Setup process

1. I installed Expo Go app on my phone and logged in after creating an account.
2. I made sure I had the latest stable version of node on my computer
```bash
node -v
npm -v
```
3. I installed Expo CLI globally
```bash
npm install -g expo-cli
```
